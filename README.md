SerenFeed

SerenFeed is a clean and minimal blog website where users can create blogs, read blogs, and connect with other bloggers. The platform is designed to make writing and reading simple, engaging, and distraction-free.

Features

Create Blogs – Share your thoughts, stories, or guides with an easy-to-use editor.

Read Blogs – Explore a variety of blogs with a clean and minimal reading experience.

Smart Break Reminder – SerenFeed gently reminds users to take a break after a set reading limit, promoting mindful and healthy screen time.

Connect with Bloggers – Interact and build meaningful connections within the blogging community.

Privacy and Control – Users can manage their profiles and stay in control of their data.

Vision

To create a mindful digital space where people can express themselves, share knowledge, and discover meaningful content without distractions.

Project Update

This project was revisited and enhanced after gaining deeper knowledge in web development. Improvements were made in design, responsiveness, and overall user experience. Through this update, strong concepts of HTML and CSS were applied, along with the integration of JavaScript to add interactivity and dynamic functionality, making the platform more engaging and user-friendly.

Tech Stack
